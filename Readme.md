# Ackerman

## Build and launch

```bash
$ cd ackerman/ackerman_robot
$ colcon build
$ . install/setup.bash
$  ros2 launch ackerman_robot ackermann_drive_example.launch.py 
```


# Diff robot

```bash
$ cd ackerman/diff_drive
$ colcon build
$ . install/setup.bash
$  ros2 launch ackerman_robot ackermann_drive_example.launch.py 
```
